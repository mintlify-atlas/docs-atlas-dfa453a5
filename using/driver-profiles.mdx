## Accessing driver profiles

You can access a driver's profile page in several ways:

- Click on a driver's name in any race results table
- Click on a driver's name in championship standings

All driver links automatically open their profile for the current season being viewed.

## Profile header

The top of each driver profile shows:

- Driver's full name
- Season summary indicator
- Breadcrumb navigation back to the season

## Current season results

The main section of the profile displays a complete race-by-race breakdown for the selected season.

### Results table

Each row in the results table shows:

- **Race** - Round number and race name, linked to the full race details
- **Constructor** - Which team the driver raced for
- **Grid** - Starting position
- **Finish** - Final race position with position change indicators
- **Status** - Race completion status or retirement reason
- **Points** - Championship points earned

<Tip>
The results are displayed chronologically by round number, making it easy to follow a driver's season progression.
</Tip>

### Position indicators

Just like in race results, green and red arrows show positions gained or lost from grid to finish:

- **Green arrows** show strong performances where the driver moved up through the field
- **Red arrows** indicate positions lost during the race

### DNF and status

Races where a driver didn't finish are clearly marked with a **DNF** tag and the reason for retirement (e.g., "Engine", "Collision", "Gearbox").

## Previous season comparison

If the driver competed in the previous season, their results from that year are automatically displayed below the current season.

This allows you to:

- Compare year-over-year performance
- See if a driver moved to a different team
- Track career progression across seasons

<Note>
Previous season data is only shown if the driver participated in that championship.
</Note>

## Switching seasons

Driver profiles are season-specific. To view a driver's performance in a different year:

1. Navigate to the desired season from the home page
2. Find the driver in race results or standings
3. Click their name to view that season's profile

Alternatively, you can change the `?season=` parameter in the URL directly.

## Mobile view

On mobile devices, the results automatically switch to a card-based layout that prioritizes the most important information:

- Finish position with position changes
- Race name and round
- Points scored
- Constructor, grid position, and status in a compact meta row

This ensures all driver statistics remain accessible on smaller screens.
