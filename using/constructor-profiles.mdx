## Accessing constructor profiles

You can view a constructor (team) profile by:

- Clicking on a constructor name in race results
- Clicking on a constructor name in championship standings

Constructor links open the profile for the season you're currently viewing.

## Profile header

Each constructor profile displays:

- Constructor's full team name
- Season summary indicator
- Breadcrumb navigation to return to the season overview

## Current season results

The profile shows all race results for both of the team's drivers throughout the season.

### Results table

Each entry in the results table includes:

- **Race** - Round number and race name, linked to full race details
- **Driver** - Which driver achieved this result, linked to their profile
- **Grid** - Starting position
- **Finish** - Final race position with change indicators
- **Status** - Race outcome or retirement reason
- **Points** - Championship points earned for the team

<Tip>
Both drivers' results appear in the same table, sorted chronologically by race and then by finish position.
</Tip>

### Grouped by race

When both drivers from the same team appear in a race, their results are visually grouped together:

- The race name appears only on the first driver's row
- The second driver's row is connected to show they're from the same race
- This makes it easy to compare teammate performance at each event

### Position changes

Green and red arrows show how many positions each driver gained or lost:

- **Green arrows** indicate positions gained during the race
- **Red arrows** show positions lost from the grid

These indicators help you quickly identify strong team performances or difficult races.

## Previous season comparison

If the constructor competed in the previous season, their full results from that year are shown below the current season data.

This comparison lets you:

- Track team development year-over-year
- See if the team retained the same drivers
- Compare performance across different seasons
- Identify trends in team competitiveness

<Note>
Previous season data only appears if the constructor participated in that championship.
</Note>

## Team insights

The constructor profile gives you a complete picture of team performance:

- **Consistency** - Compare results between both drivers at each race
- **Reliability** - See which races resulted in DNFs and why
- **Points scoring** - Track which races and drivers contributed to the championship
- **Driver comparison** - Evaluate how teammates performed against each other

## Switching seasons

To view a constructor's profile in a different season:

1. Navigate to your desired season from the home page
2. Locate the constructor in race results or standings tables
3. Click the constructor name to view that season's profile

You can also modify the `?season=` URL parameter directly.

## Mobile view

On smaller screens, constructor results switch to an optimized card layout:

- Finish position with position change indicators
- Driver name linked to their profile
- Points earned
- Race information, grid position, and status in a compact format

This ensures team performance data remains clear and accessible on all devices.
